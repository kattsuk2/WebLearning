レイアウトによく使うスタイル

width


<div> width指定なし </div>
<div style="width: 200px"> width 200px指定 </div>
<div style="width: 70%"> width 70％指定 </div>
<div style="width: 100%"> width 100％指定 </div>

結果

width指定なし
width: 200px
width: 70%
width: 100%

margin

  • margin-top
  • margin-bottom
  • margin-left
  • margin-right

上下左右に一気にmarginを指定（一気に指定できるものは並べる順番は全て同じ）

margin: 10px(top) 5px(right) 10px(bottom) 5px(left)

上下同じでと左右同じを一気にmarginを指定（一気に指定できるものは並べる順番は全て同じ）

margin: 10px(top,bottom) 5px(left,right)

上下別でと左右同じを一気にmarginを指定（一気に指定できるものは並べる順番は全て同じ）

margin: 10px(top) 5px(left,right) 20px(bottom)


ボックスの左側にmarginを入れる
<div> width: 500px margin-leftなし </div>
<div style="width: 500px; margin-left: 10px"> width 500px指定 margin-left: 10px</div>
<div style="width: 500px; margin-left: 10%"> width 500px指定 margin-left: 10% </div>

結果

width: 500px
width: 500px margin-left: 10px
width: 500px margin-left: 10%

padding（box-sizingに注意！後で説明、ここでは、border-box）

  • padding-top
  • padding-bottom
  • padding-left
  • padding-right

ボックスの左側にpaddingを入れる
<div> width: 500px padding-leftなし </div>
<div style="width: 500px; padding-left: 10px"> width 500px指定 padding-left: 10px</div>
<div style="width: 500px; padding-left: 10%"> width 500px指定 padding-left: 10% </div>

結果

width: 500px
width: 500px padding-left: 10px
width: 500px padding-left: 10%

border（box-sizingに注意！後で説明、ここでは、border-box）

  • border-top
  • border-bottom
  • border-left
  • border-right

<div> width: 500px border-top: 1px solid blue </div>
<div> width: 500px border-right: 1px solid blue </div>
<div> width: 500px border-bottom: 1px solid blue </div>
<div> width: 500px border-left: 1px solid blue </div>
<div> width: 500px border: 1px solid blue </div>

結果

width: 500px; border-top: 1px solid blue
width: 500px; border-right: 1px solid blue
width: 500px; border-bottom: 1px solid blue
width: 500px; border: 1px solid blue

float


<div> width: 200px float: left </div>
<div> width: 200px </div>
<div> width: 200px </div>
<div> width: 200px </div>
<div> width: 200px </div>

width: 200px; float: left
width: 200px;
width: 200px;
width: 200px;
width: 200px;

clear


<div> width: 200px float: left </div>
<div> width: 200px </div>
<div> width: 200px clear: both</div>
<div> width: 200px </div>
<div> width: 200px </div>

結果

width: 200px; float: left
width: 200px;
width: 200px; clear: both
width: 200px;

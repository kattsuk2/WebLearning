HyperText Markup Languageの略

つまり、ハイパーなテキストをマークアップするための言語なんです。じゃあ、ハイパーなテキストって？って話になるわけです。そこは、簡単に言うと文書と文書を関連付ける仕組みってことです。それでテキストを超えるってことでハイパーテキストってことにしといてください。aタグのリンクのことをハイパーリンクっていいますよね？

HTML5までの道のり

  • HTML1.0、HTML+
  • HTML2.0
  • HTML3.0、HTML3.2
  • HTML4.0、HTML4.1
  • ISO/IEC 15445:2000

XHTMLはってなるかもなので、一応HTML4.0をもとにしてXMLの文法で定義したものです。ざっくりすぎかもですけど（Extensible HyperText Markup Language）

  • XHTML1.0
  • XHTML Basic
  • XHTML Modularization (Modularization of XHTML, M12n)
  • XHTML1.1
  • XHTML1.2
  • XHTML2.0
  • XHTML5

って流れで変わってきたんですね。。。結構XHTML1.0、1.1とかは使った人が多いかもですね。

HTML5の最近・・

HTML5はWeb Hypertext Application Technology Working Groupによって2004年に定められたWeb Applications 1.0にWeb Forms 2.0を取り入れたものがWorld Wide Web Consortiumの専門委員会に採用され、World Wide Web Consortiumより2008年1月22日にドラフト（草案）が発表され、2014年10月28日に HTML5 が勧告され、2016年11月1日に HTML 5.1 が勧告され、2017年12月14日に HTML 5.2 が勧告された。

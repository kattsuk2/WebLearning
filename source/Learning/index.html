CSS3

CSS(Cascading Style Sheet）は、マークアップされたものHTML、XTHML、XML、SVGなどの文章の体裁や見栄えを表現するためのスタイルシート

CSS書き方


h1 { color: red } /* セレクタ { プロパティ: 値 } */

h2 {
  color: blue; /* 複数項目プロパティを設定するときは、セミコロンで閉める */
  background-color: red;
}
h2 {color: blue; background-color: red;} /* まぁ上と一緒だけど一応ね */

/* コメントは/*で始まり*/で終わる */
/*
h2 {
  color: red;
}
*/ 複数行も同じコメントにしたい個所を囲む

CSSを書く場所

CSSはいろんな所にかけてしまいます。HTMLのheadタグ内、bodyタグ内またはタグに直接書く、CSSファイルを作成する。といった感じで書こうと思えば簡単に書けます。


            HTMLファイル
<!DOCTYPE html>
<head>
<link rel="stylesheet" href="style.css">
<title>TEST</title>
<style type="text/css">
div {
  color: red;
}
</style>
</head>
<body>
<style>
div {
  color: blue;
}
</style>

<div style="color: black">
  <p>コンテンツ</p>
</div>
</body>
</html>
          

/* style.css */
div {
  color: white;
}

上の例のようにいたるところに同じセレクタに対して書くことができます。

このように書かれた場合に最終的には、どのスタイルが適用されるのでしょう。。。

最終的には、style="color: black"とdivタグに直接書かれているものが適用されます。スタイルは基本あと勝ちです。ただし、!importantを値の後ろにつけることで強制的に聞かせることができます。しかしこれを使うとこれ以降にcolorプロパティがあっても適用できません。!importantは気を付けて使いましょう。


/* style.css */
div {
  color: white !important;
}
